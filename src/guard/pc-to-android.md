# 从电脑导入文件到Android手机App

本文介绍如何将 Windows 或 Mac 电脑中的文件传输到Android手机App。

## 一、Windows 电脑

### 操作步骤

1. 手机通过 USB 连接 Windows，下拉状态栏，修改"USB连接方式"为"文件传输"
2. 进入 Windows 的资源管理器，找到手机对应的设备，进入 `/Android/data/com.pollykann/files/LocalResources` 目录
3. 将 Windows 电脑中的视频和字幕文件拷贝到此目录即可

## 二、苹果电脑（Mac）

### 操作步骤

1. Mac 安装"Android File Transfer"应用
2. 手机通过 USB 连接 Mac，Android File Transfer 将自动打开。如果没有自动打开，手动点击打开
3. 在 Android File Transfer 中进入 `/Android/data/com.pollykann/files/LocalResources` 目录
4. 将 Mac 电脑中的视频和字幕文件拖拽到此目录即可

## 三、注意事项

> ⚠️ **重要提示**：应用卸载后，导入的数据将丢失，重要数据请另外备份！
